Metodologi & Atribusi Proyek

Pencapaian Kriteria Evaluasi

  • Akurasi Matematis & Fisis Teruji: Formulasi ΔE = 2s_{i,j} ∑ s_tetangga berhasil diimplementasikan. Kondisi batas periodik (periodic boundary) dijaga keamanannya menggunakan operasi modulo pada indeks array matriks kisi. Fungsi probabilitas penerimaan Metropolis R = e^{-ΔE/T} terbukti akurat memicu perpecahan simetri spontan pada suhu rendah.
  • Efisiensi Komputasi Optimal: Algoritma terbebas dari perhitungan ulang energi total sistem (Hamiltonian) pada setiap step iterasi. Komputasi murni difokuskan pada kalkulasi perbedaan selisih energi lokal. Struktur kodingan ditulis rapi dengan dokumentasi komentar informatif untuk memudahkan code review.
  • Komunikasi Ilmiah Berkualitas: Seluruh grafik visualisasi (peta domain magnetik & kurva magnetisasi termal) memenuhi standar penyajian data dengan menyertakan atribut absis, ordinat, judul, dan colorbar. Laporan ini juga menyajikan analisis mendalam terkait interaksi energi pertukaran vs. agitasi termal di tiap kasus fasa.
